Where Sideline goes next

Today it's a hackathon prototype. Here's the roadmap from AI referee to autonomous sports intelligence platform.

Phase 1 — Today
Agentic Referee MVP
Cloud VLM inference via Nebius Token Factory. Single camera. Tennis. Tool calling for score, calls, and robot gestures. MCP + A2A protocols. Dashboard + MentorPi robot.
psychology

Cloud Inference

Nebius Token Factory VLMs. 1-2s per frame. Good enough for between-point calls.

smart_toy

Single Robot

MentorPi speaks calls, gestures with arm, tracks play with camera.

sports_tennis

Tennis First

Clear rules, visible calls, structured scoring. Perfect proving ground.

Phase 2 — Edge Deployment
Real-time on-device inference
Move inference to edge hardware. Sub-100ms latency for in-rally tracking. NVIDIA Jetson AGX Orin running quantized models. No cloud dependency during gameplay.
bolt

Jetson AGX Orin

Run Cosmos Reason 2 locally. 3-18s full reasoning, sub-100ms for tracking-only mode.

phone_iphone

Phone as Referee

Camera in, voice out. No hardware needed. Just point your phone at the court.

sync

Hybrid Mode

Edge for tracking, cloud for deep reasoning on disputed calls. Best of both.

Phase 3 — World Action Models
Zero-shot referee policies from video
Train a World Action Model (like NVIDIA's DreamZero) that jointly predicts future game states and referee actions from video. The model learns physics, rules, and judgment from watching thousands of matches — then referees new sports zero-shot.
public

DreamZero-style WAM

Video diffusion backbone jointly predicting world states + actions. 14B parameter model achieving 7Hz real-time control.

NVIDIA Research, 2026
target

Zero-shot Transfer

Train on tennis, deploy on badminton — no retraining. The model understands physics, not just rules.

fitness_center

NVIDIA Osmo

Orchestrate training: synthetic data → model training → RL fine-tuning → hardware-in-the-loop testing.

Phase 4 — Autonomous Sports Intelligence
The humanoid referee walks the court
A fully autonomous humanoid referee that perceives the game, walks the court, makes calls, explains decisions, and communicates with players. Multi-agent coordination across cameras, robots, and cloud. Every amateur match, every sport, everywhere.
directions_run

Humanoid Referee

Unitree G1 or similar. Walks sideline, signals with arms, announces with voice, tracks ball with head.

hub

Multi-Agent Mesh

3+ camera agents, coordinator, robot agent, commentary agent — all via A2A. Autonomous VAR.

public

Every Sport

Cricket, pickleball, badminton, table tennis, soccer. Same platform, sport-specific rules engine.

analytics

Sports Analytics

Every call generates data. Player analytics, performance trends, coaching insights — built in.
