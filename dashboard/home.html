Action Protocols

Three ways to act

The same agent brain outputs through function calls, MCP tools, and agent-to-agent communication.

Function Calls

OpenAI Tool Calling

The VLM decides which tools to invoke — update_score, announce_call, robot_gesture. Native tool calling through Nebius Token Factory.

MCP

Model Context Protocol

Expose referee capabilities as standardized MCP tools. Any AI client — Claude, ChatGPT, Cursor — can use Sideline as a referee service.

A2A

Agent-to-Agent Protocol

Multiple Sideline agents coordinate via Google's A2A protocol. Three cameras, three agents, one coordinator — autonomous VAR.
