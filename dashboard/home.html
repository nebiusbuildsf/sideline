An Agent, Not a Classifier

Sideline maintains state across an entire match. It remembers every call, tracks every point, and reasons about context — not just pixels.

visibility

Perceive

Multiple sensor inputs — court cameras, robot vision, depth sensors, phone cameras. Frames extracted and fused for the vision model.

psychology

Reason

Nebius Token Factory VLM with chain-of-thought: what happened → which rule applies → what's the call → confidence score. Full reasoning visible.

precision_manufacturing

Act

Tool calls update scores, announce calls, signal gestures. MCP exposes tools externally. A2A coordinates multiple agents. Actuators execute physically.

Three Protocols, One Brain

The agent dispatches actions through layered protocols. Each adds capability without replacing the one below.

bolt

Function Calls

Native tool_use via Nebius API. The VLM decides which tools to invoke — update_score, announce_call, robot_gesture.

extension

MCP Server

Expose referee tools via Model Context Protocol. Any MCP client — Claude, ChatGPT, Cursor — can use Sideline as a tool.

hub

A2A Protocol

Agent-to-agent coordination. Multiple cameras, multiple agents, one coordinator — like autonomous VAR.
